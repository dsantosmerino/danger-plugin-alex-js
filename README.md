> Catch insensitive, inconsiderate writing with alex.

## Usage

Install:

```sh
yarn add danger-plugin-alex-js --dev
```

At a glance:

```js
// dangerfile.js
import alexJs from 'danger-plugin-alex-js'

alexJs()
```
